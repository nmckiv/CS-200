Short Paragraph

This project is for the development of the data processing software for Chocoholics Anonymous, or ChocAn for short. ChocAn is an organization dedicated to helping people addicted to chocolate. Our team consists of John Byrd, Girwan Dhakal, Ben Haas, Pearson Hunter, Nolan McKivergan, and Rowan Romanauskas. We are dividing this Project into 4 Sub-Projects, the first of which being Project 1. As a whole, this project will be completed over the next 4 months (the semester) and will be coded in both java and HTML. For the first project, our submitter is Pearson. Our current tasks consist of creating a glossary, a use case diagram, use case descriptions for each of the use cases in our diagram, a story selection matrix, two generalized security user stories, and this short paragraph. Our group tackled these tasks by first creating individual use case diagrams and glossaries, as can be seen in the individual team member report files, and then meeting to discuss the rest of the project. When we met, we came up with a final use case diagram (seen in the use case diagram tab of the header) and assigned each member a set of tasks to complete. Each member handled their own task and uploaded them to Bitbucket, where Pearson took all of them and assembled the information into the team report files. As for who did what, please refer to the above task distribution table.
