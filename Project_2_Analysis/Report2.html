System Operations Paragraph

In order for the system to operate, all of the system will be computerized. Most of the functionality will come from the program and being able to access ChocAn data centers. All of the data will be stored within these data centers. The hardware that will be needed is anything in the data centers, for example hard drives and everything needed to keep those online 24/7. Also, in order to access the program, any actor in the system will need a computer terminal of their own. The ChocAn system will interact with others as a program that exchanges data. The only need for interaction is to share files or update information. Users will interact with the system by logging in and clicking through the necessary menus to accomplish the tasks they need.
